Garden Planner

GardenBedExample.java
GardenBed.java
FlowerBed.java

To run the program, please go to the GardenBedExample.java file and press 'run'. It will 
generate a JOptionPane menu dialog containing a menu:
1. Add garden bed        - create a new garden bed from user input
2. Remove garden bed     - delete a garden bed that has been created 
3. Print garden list     - print the list of garden beds
4. Print cost report     - print the cost report for all garden beds
5. Exit
